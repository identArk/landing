n8n quickstart

v1.0.1 on npm
  1. In n8n: Settings → Community Nodes → Install.
  2. npm package name: n8n-nodes-identark. Check "I understand the risks".
  3. Add an IdentArk node to any workflow. Resource = LLM, Operation = Invoke.
  4. Credential → Create new → paste your IdentArk API key, leave Control Plane URL as https://identark-cloud.fly.dev.
  5. Click Test step. You should see an LLM response with cost + risk score attached.

Direct API quickstart

curl -X POST https://identark-cloud.fly.dev/v1/llm/invoke \
  -H "Authorization: Bearer $IDENTARK_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "new_messages": [{"role":"user","content":"Hello, who are you?"}],
    "model": "gpt-4o"
  }'

Use this from any language. The gateway holds the LLM provider key; your code only ever holds the IdentArk session token.
